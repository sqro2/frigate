Frigate v1.0 User Guide

About Frigate

Frigate is a html preprocessor which helps write html faster with much less code,time,effort.

Syntex for writing basic markup

					  element (.class | #id)*multiplier

Example:

Input:

				       div(.class-name #id-name)*6

Output:

						<div id="id-name" class="class-name"> </div>
						<div id="id-name" class="class-name"> </div>
						<div id="id-name" class="class-name"> </div>
						<div id="id-name" class="class-name"> </div>
						<div id="id-name" class="class-name"> </div>
						<div id="id-name" class="class-name"> </div>

Note -

* The "id" and "class" selectors are optional.They can be null.

* Each element can have max one "id" and one "class".

* The "multiplier" can only be a postitive integer and it's optional.

Example:

Input:

                        div*6

output: 

						<div id="null" class="null"> </div>
						<div id="null" class="null"> </div>
						<div id="null" class="null"> </div>
						<div id="null" class="null"> </div>
						<div id="null" class="null"> </div>
						<div id="null" class="null"> </div>

Syntex for Nesting markup

					element1(.class #id){
						   element2(.class #id){
							   element3(.class #id){
									......so on
								  }*multiplier   
						   }*multiplier
					}*multiplier

Example:

Input:

					header(.header-class #header-id){
						  nav(.nav-class #nav-id){
							   ul(.ul-class #ul-id){
								   li(.li-class #li-id){
									   a
									}*5
							   }*2
						  }
					}

Output:

Note -

* The "id" and "class" selectors are optional.They can be null.
